# 霍尔传感器接线检查清单

## 🔴 紧急检查（上电前）

### 1. 霍尔模块供电
- [ ] **测量霍尔模块 VCC 引脚**
  - 预期：3.3V
  - 如果显示 5V → **立即断电并修改接线**

- [ ] **测量霍尔模块 GND 引脚**
  - 预期：0V（与 STM32 AGND 连通）

### 2. 霍尔输出电平
- [ ] **霍尔输出类型确认**
  - 查看霍尔模块规格书
  - 如果是 **开漏(OD)** → 需要上拉电阻到 3.3V
  - 如果是 **推挽(PP)** → 直接连接

### 3. STM32 GPIO 配置
- [ ] **确认 GPIO 模式为输入**
  - PB6, PB7, PB8 应该是 `GPIO_MODE_INPUT`
  - 不要配置为 `GPIO_MODE_ALTERNATE`（TIM4 编码器模式）

## 📊 正常状态参考

| 引脚 | 电机静止 | 电机旋转 | 说明 |
|------|---------|---------|------|
| PB6 (HA) | 0V 或 3.3V | 0V ↔ 3.3V 跳变 | 霍尔 A 相信号 |
| PB7 (HB) | 0V 或 3.3V | 0V ↔ 3.3V 跳变 | 霍尔 B 相信号 |
| PB8 (HC) | 0V 或 3.3V | 0V ↔ 3.3V 跳变 | 霍尔 C 相信号 |

## 🛠️ 常见修复方案

### 方案 A：霍尔供电从 5V 改为 3.3V
```
原来：霍尔 VCC → 5V
修改：霍尔 VCC → 3.3V（从 STM32 引脚或 LDO 输出）
```

### 方案 B：添加上拉电阻（如果是开漏输出）
```
霍尔输出 → 上拉电阻 4.7kΩ → 3.3V
霍尔输出 → STM32 GPIO (PB6/PB7/PB8)
```

### 方案 C：检查 HC 相接线
```
PB8 应该是高阻态输入
如果 0.56V，可能是：
  - HC 相接线松动
  - HC 霍尔模块损坏
  - 需要重新焊接
```

## ⚡ 快速测试步骤

1. **断电**
2. **测量霍尔模块 VCC** → 确认是 3.3V
3. **重新上电**
4. **再次测量 PB6/PB7/PB8**
5. **手动旋转电机轴**
6. **观察电压是否跳变**
