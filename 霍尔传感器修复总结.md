## 📊 数据流

```
TIM4 中断 (每换相一次)
  └─ HALL_Get_Electrical_Angle()
      ├─ 读取 CCR1 捕获值
      ├─ 计算 AvrElSpeedDpp（电角速度）
      ├─ 计算 HallSpeed（机械转速 rpm）
      ├─ GPIO 读取 → 扇区判断 → MeasuredElAngle
      └─ FirstOrderRC_LPF() 滤波

ADC 中断 (20kHz)
  ├─ HallElAngle += AvrElSpeedDpp（角度累加）
  ├─ 一圈校准（同步实测角度）
  ├─ 同步到 hall_sensor（供 bsp_hall 接口）
  └─ 更新 foc.rotor_angle / foc.rotor_speed

MiniFOC_CurrentLoop() (40kHz)
  ├─ 读取 foc.rotor_angle
  ├─ 读取 foc.rotor_speed
  └─ 电流环 PID 控制
```

## ⚙️ 工作原理

### TIM4 中断（换相时刻）
```
捕获值 → 计算转速 → 更新 AvrElSpeedDpp
GPIO 读取 → 扇区判断 → 更新 MeasuredElAngle
滤波 → 更新 HallSpeed
```

### ADC 中断（20kHz 平滑累加）
```
HallElAngle += AvrElSpeedDpp  // 线性累加，平滑
一圈校准                     // 消除累积误差
同步到 foc/hall_sensor        // 供 MiniFOC 使用
```

## 🎯 修复的问题

| 问题 | 原因 | 解决方案 |
|------|------|---------|
| **启动卡顿** | HallElAngle 初始化为0或随机值 | 查表映射到扇区中心角 |
| **角度跳变** | HallElAngle 只在换相时更新 | ADC 20kHz 平滑累加 |
| **累积误差** | 长时间运行角度漂移 | 一圈校准，同步实测值 |
