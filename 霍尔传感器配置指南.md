# 霍尔传感器配置指南

## 📍 引脚定义

使用以下3个引脚连接霍尔传感器：

| 引脚 | 信号 | 说明 |
|------|------|------|
| PB12 | HALL_U | U相霍尔信号 |
| PB3  | HALL_V | V相霍尔信号 |
| PB4  | HALL_W | W相霍尔信号 |

**接线说明**：
- 这些引脚接电机霍尔传感器的输出（开漏/推挽）
- 需要外部上拉电阻（10kΩ）或使用内部上拉
- 霍尔传感器供电：3.3V 或 5V（根据传感器规格）

---

## 🔧 CubeMX配置步骤

### 步骤1：打开 .ioc 文件
1. 打开 `My_FOC.ioc`
2. 进入 **Pinout & Configuration** 标签页

### 步骤2：配置HALL_U (PB12)
1. 点击 **PB12** 引脚
2. 选择模式：**GPIO_Input**
3. GPIO参数设置：
   - **GPIO_PuPd**: GPIO_PULLUP（内部上拉）
   - **GPIO_Label**: HALL_U
4. 点击 **Apply**

### 步骤3：配置HALL_V (PB3)
1. 点击 **PB3** 引脚
2. 选择模式：**GPIO_Input**
3. GPIO参数设置：
   - **GPIO_PuPd**: GPIO_PULLUP（内部上拉）
   - **GPIO_Label**: HALL_V
4. 点击 **Apply**

### 步骤4：配置HALL_W (PB4)
1. 点击 **PB4** 引脚
2. 选择模式：**GPIO_Input**
3. GPIO参数设置：
   - **GPIO_PuPd**: GPIO_PULLUP（内部上拉）
   - **GPIO_Label**: HALL_W
4. 点击 **Apply**

### 步骤5：保存并生成代码
1. 按 **Ctrl + S** 保存
2. 点击 **Project Manager** → **Generate Code**
3. 等待代码生成完成

---

## 📝 配置完成后

生成的代码将包含：
- `HALL_U_Pin`, `HALL_U_GPIO_Port`
- `HALL_V_Pin`, `HALL_V_GPIO_Port`
- `HALL_W_Pin`, `HALL_W_GPIO_Port`

这些宏定义将在 `main.h` 或 `gpio.h` 中。
