# 直流无刷电机FOC驱动器 - 硬件排查指南

## 📍 问题现象

**现象：**
- ✅ V相电流 (I1) 有微小波动 (~0.005A)
- ❌ U相电流 (I0) 完全不动 (0.000A)
- ❌ W相电流 (I2) 完全不动 (0.000A)
- ✅ 电机不转（正常，程序默认关闭）
- ✅ V相电压 (I3) 有读数

**问题分析：**
ADC1 的 U相和 W相数据完全没有更新，只有 ADC2 的 V相有数据。

## 📋 常见问题原因

| 现象 | 可能原因 |
|------|---------|
| U/W相完全不动 (0.000A) | OPAMP1/3 未工作 |
| V相有波动但数值很小 | 零点校准正常，OPAMP2 工作 |
| 三相都完全不动 | ADC 未触发/供电异常 |
| 三相都有波动但不对称 | 硬件接线错误/增益不匹配 |

---

## 💡 临时解决方案

**在排查硬件期间，可以先测试 V相单相系统：**

修改 `bsp_adc.c`：
```c
/* 临时方案：只用 V相 */
CurrlValue[0] = 0.0f;  /* U相 = 0 */
CurrlValue[1] = (float)(ADC_Data[1] - offset[1]) * 0.021972f;  /* V相 */
CurrlValue[2] = -CurrlValue[1];  /* W相 = -V相（估算）*/
```

**注意：** 这只是临时方案，**必须修复硬件问题才能实现真正的三相FOC**。
